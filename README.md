# Інтерактивна сторінка реєстрації

Навчальний веб-проєкт, створений з використанням HTML5, CSS3 та JavaScript.

## Опис проєкту

Даний проєкт являє собою інтерактивну сторінку реєстрації учасників конференції.

Користувач може:

- Ввести ПІБ
- Ввести електронну пошту
- Ввести номер телефону
- Обрати курс навчання
- Надіслати заявку без перезавантаження сторінки

Після успішного заповнення форми користувач отримує повідомлення про успішну реєстрацію.

---

## Функціонал

- Семантична HTML5-розмітка
- Адаптивний дизайн

- Перевірка номера телефону
- Обробка подій JavaScript
