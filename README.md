# Інтерактивна сторінка реєстрації

Практична робота з HTML5, CSS3 та JavaScript.

## Функціонал

- Реєстрація учасника конференції
- Перевірка введених даних
- Валідація номера телефону
- Адаптивний дизайн
- Робота без перезавантаження сторінки

## Технології

- HTML5
- CSS3 (Flexbox)
- JavaScript
